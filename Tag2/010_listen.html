Listen in HTML

In diesem Abschnitt kann ich alles über verschiedene Listen nachlesen. HTML kennt:
<ol> (sortierte Listen), <ul> (unsortierte Listen) und <dl> (beschreibende Listen).

1. Wann nutzen wir welche Liste?

  • <ol>: Wenn die Reihenfolge wichtig ist (Bsp.: Rangliste, Anleitung, Rezept)
  • <ul>: Wenn die Reihenfolge egal ist (Bsp.: Aufzählung, Einkaufsliste, To-Do Liste)
  • <dl>: Für Begriffe mit Erklärung (Bsp.: Glossar, FAQ, Wörterbuch)

2. sortierte Liste (<ol>)

Eine sortierte Liste nummeriert die Einträge automatisch

Besipiel (früh aufstehen)

  1. Augen öffnen
  2. Aufrichten
  3. Uns aus dem Bett bewegen

Wichtige Attribute

  • type: 1, A, a, I, i
  • start: Startwert festlegen
  • reversed: Nummerierung umkehren

Besipiel (Tiere):

  1. Hund
  2. Katze
  3. Kuh
  4. Elephant
  5. Meerschweinchen
  1. Hund
  2. Katze
  3. Kuh
  4. Elephant
  5. Meerschweinchen
  1. Hund
  2. Katze
  3. Kuh
  4. Elephant
  5. Meerschweinchen

3. Unsortierte Liste (<ul>)

Eine unsortierte Liste stellt Aufzählungen dar, bei der die Reihenfolge egal ist.

Beispiel (Städte):

  • Madrid
  • London
  • Paris

Wichtige Attribute:

  • type: none, circle, square

Bespiel (Städte)

  • Madrid
  • London
  • Paris
  • Madrid
  • London
  • Paris
  • Madrid
  • London
  • Paris

4. Beschreibende Listen (<dl> = Description List)

Eine beschreibende Liste beateht aus <dt>(Begriff) und <dd> (Description = Beschreibung)

Beispiel (Begriffe in der IT):

HTML
Hypertext Markup Language
Auszeichnungssprache zur Strukturierung der Webseite

CSS
Cascading Style Sheets
Sprache für das Design und Layout der Webseite

PHP
Hypertext Preprocessor. Ursprünglich stand es für „Personal Home Page“
serverseitige Skriptsprache

5. Verschachtelte Listen

Listen können in anderen Listen eingebettet werden. (Bsp. Menüs oder Kategorien).

Beispiel (Elektrogeräte):

  • Produkt
    • Handys
      • Android
        • Samsung
        • Huawei
        • Xaomi
      • iOS
        • Apple

Anleitung: Verschachtelte Listen in HTML

  1. Erstelle zuerst eine normale Liste mit <ul> oder <ol>.
  2. Lege einen Listeneintrag mit <li> fest.
  3. Schreibe den Text für den Haupteintrag in dieses <li>-tag.
  4. Öffne innerhalb des selben <li>-tags eine neue Liste (mit <ul> oder <ol>).
  5. Lege in dieser neuen Liste die Unterpunkte mit <li> an.
  6. Schließe zuerst die Unterliste und danach das äußere <li>.
